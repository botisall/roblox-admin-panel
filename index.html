  1. Buka Roblox Studio
  2. Buka game Anda
  3. Masuk ke ServerScriptService
  4. Buat Script baru, beri nama: APIConnector
  5. Copy code di bawah ini ke script tersebut:

🔗 Step 2: Import Data dari Roblox

  1. Jalankan game Anda di Roblox Studio
  2. Join game sebagai admin
  3. Ketik command di chat: :exportdata
  4. Tekan F9 untuk buka Console
  5. Copy semua data JSON yang muncul
  6. Paste di textarea di bawah ini:

💡 Tips:

  • Website ini menggunakan memory storage untuk simpan data
  • Data akan realtime sync dengan API server
  • Gunakan command :exportdata untuk export data dari game
  • Gunakan command :syncnow untuk manual sync
